## 1. 简介

TensorLogic是一种编程语言，将张量代数与逻辑编程统一起来，实现神经符号AI。

### 主要特性

- **张量运算**: GPU加速的高性能计算
- **自动微分**: 内置梯度计算
- **学习系统**: 多种优化器的梯度下降
- **逻辑编程**: 关系、规则和查询
- **神经符号集成**: 实体和关系的嵌入

## 2. 程序结构

```tensorlogic
// 声明
tensor w: float32[10] learnable = [...]
relation Parent(x: entity, y: entity)

// 主执行块
main {
    result := w * w

    learn {
        objective: result,
        optimizer: sgd(lr: 0.1),
        epochs: 50
    }
}
```

## 3. 数据类型

| 类型 | 描述 | 精度 |
|------|-------------|-----------|
| `float32` | 32位浮点数 | 单精度 |

| `int32` | 32位整数 | 有符号整数 |
| `int64` | 64位整数 | 长整数 |
